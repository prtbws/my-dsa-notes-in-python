# My DSA Notes in Python

This repository serves as my personal DSA knowledge base, built while learning Competitive Programming, Coding Interviews, and Algorithmic Problem Solving.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/prtbws/my-dsa-notes-in-python.git
```

### Open in VS Code

```bash
cd my-dsa-notes-in-python
code .
```

### Run Python Files

```bash
python filename.py
```

or

```bash
python3 filename.py
```
